무역 데이터 분석

1. 무역 데이터 링크

  • UN Comtrade: UN의 글로벌 무역 데이터
  • WTO 통계: WTO(세계무역기구) 통계
  • Trade Map: 무역 데이터와 시각화 도구 제공

2. 무역 데이터 API로 추출

다양한 플랫폼에서 제공하는 API를 통해 무역 데이터를 자동으로 추출할 수 있습니다:

  • UN Comtrade API: 글로벌 무역 데이터 API
  • Trade Map API: 무역 데이터 조회용 API
  • WTO 데이터 API: 세계 무역 관련 데이터 API

예시 코드:

fetch('https://comtrade.un.org/api/get?type=C&freq=A&px=HS&ps=2022&r=all&p=0&rg=all&cc=TOTAL')
    .then(response => response.json())
    .then(data => console.log(data));

3. 무역 데이터 분석 예제

무역 데이터를 활용한 분석 예제:

  • 수출입 데이터 시각화
  • 국가별 무역 트렌드 분석
  • 상품군별 주요 교역 국가 비교

Python 예제:

import pandas as pd
import matplotlib.pyplot as plt

# 데이터 읽기
data = pd.read_csv('trade_data.csv')

# 상품별 수출 트렌드 시각화
data.groupby('Year')['Export Value'].sum().plot(kind='line')
plt.title('연도별 수출액')
plt.xlabel('연도')
plt.ylabel('수출액')
plt.show()
